# Sentiment Analysis of Twitter Data

##  Overview
Analyzed Twitter data to understand public sentiment trends using text processing techniques.

---

##  Dataset
- 146,000+ tweets containing keyword 'Trump'

---

##  Tools Used
- Python  
- Pandas, NumPy  

---

##  Key Analysis
- Text cleaning and preprocessing  
- Stop word removal  
- Sentiment classification  

---

##  Key Insights
- Positive-to-negative sentiment ratio: 1.34:1  
- Overall sentiment leaned positive  
- Processed ~59,500 stop words (40.87%)  

---

##  Business Impact
- Helps understand public opinion trends  
- Supports decision-making for media and businesses
