# Sales & Distribution Analytics Dashboard

##  Overview
Developed interactive Tableau dashboards to analyze sales performance across products, regions, and factories.

---

##  Tools Used
- Tableau

---

##  Download Dashboard
You can download and explore the interactive Tableau dashboard using the `.twbx` file in this repository.

---

##  Key Features
- Interactive dashboards with filters and parameters  
- KPI tracking (sales, profit, units)  
- Regional and product-level analysis  

---

##  Key Insights
- 60%+ revenue concentrated in coastal regions  
- Strong seasonal trends with peak sales in Q4  
- High dependency on top-performing products  
- High proportion of one-time buyers
